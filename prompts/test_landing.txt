Create a modern, responsive landing page for a tech company with the following features:

1. **Header Section:**
   - Company logo and name "TechCorp"
   - Navigation menu with Home, Services, About, Contact
   - Hero section with compelling headline and call-to-action button

2. **Main Content:**
   - Services section with 3 service cards
   - About section with company description
   - Contact form with name, email, and message fields

3. **Styling Requirements:**
   - Modern CSS with gradients and shadows
   - Responsive design for mobile and desktop
   - Smooth animations and hover effects
   - Professional color scheme (blues, whites, grays)
   - Clean typography with good spacing

4. **Footer:**
   - Social media links
   - Copyright information
   - Contact details

Make it visually appealing and professional. Use modern CSS techniques like flexbox, grid, and CSS variables.
